# Phishing-Website-Detetction-using-URL-Based-Features-and-Ensemble-Learning-Models

• Developed a machine learning model to detect phishing websites using URL-based features.

• Trained and evaluated 7 classification algorithms: K-Nearest Neighbors, Logistic Regression, NaiveBayes, Decision Tree, Random Forest, Support Vector Machine, and XGBoost.

• Achieved 96.99% accuracy, with XGBoost performing as the best model.

• Built an interactive web dashboard using Streamlit to demonstrate real-time prediction results.

• Implemented the project using Python, scikit-learn, pandas, matplotlib, and Streamlit
